# Moodboards Campbnb Québec

Collection de références visuelles et d'inspiration pour le design Campbnb Québec.

## Identité visuelle

### Couleurs du Québec

L'identité visuelle s'inspire des paysages québécois:

- **Vert forêt**: Les immenses forêts boréales
- **Bleu lac**: Les nombreux lacs et rivières
- **Beige bois**: Les chalets et cabanes en bois
- **Terre**: Les sols et roches caractéristiques

### Ambiance

- **Chaleureux**: Accueil québécois authentique
- **Naturel**: Connexion avec la nature
- **Moderne**: Design épuré et contemporain
- **Accessible**: Simple et intuitif

## Références visuelles

### Paysages québécois

#### Forêts boréales
- Couleurs: Verts profonds (#2D572C), verts moyens (#4A7A4A)
- Texture: Feuillages denses, lumière filtrée
- Ambiance: Mystérieux, apaisant

#### Lacs et rivières
- Couleurs: Bleus clairs (#3B8EA5), bleus profonds (#2A6B7F)
- Texture: Eau calme, reflets
- Ambiance: Serein, rafraîchissant

#### Chalets et cabanes
- Couleurs: Beiges (#F5E5D5), bruns chauds
- Texture: Bois naturel, pierre
- Ambiance: Rustique, confortable

### Illustrations

#### Style recommandé
- **Ligne claire**: Contours nets, couleurs plates
- **Minimaliste**: Détails essentiels seulement
- **Couleurs**: Palette limitée (primary, secondary, accent)
- **Formes**: Arrondies, organiques

#### Éléments à illustrer
- Tentes
- Chalets
- Feux de camp
- Canots
- Arbres (sapins, érables)
- Animaux (orignaux, castors, oiseaux)
- Équipements de camping

## Références design

### Applications similaires

#### Airbnb
- **Points forts**: Navigation claire, cards élégantes, photos mises en avant
- **À adapter**: Couleurs québécoises, typographie Plus Jakarta Sans

#### Hipcamp
- **Points forts**: Focus nature, filtres avancés
- **À adapter**: Interface plus moderne, intégration IA

### Design Systems

#### Material Design 3
- **Utilisé pour**: Principes de navigation, animations, accessibilité
- **Adapté avec**: Couleurs et typographie custom

#### Human Interface Guidelines (iOS)
- **Utilisé pour**: Principes de clarté, hiérarchie visuelle
- **Adapté avec**: Style Android-first avec cohérence iOS

### Écrans clés

1. **Welcome Screen**
- Image de fond: Lac québécois au lever du soleil
- Overlay: Dégradé sombre pour lisibilité
- Logo: Icône feu de camp ou tente
- Boutons: Primary et outlined

2. **Home Screen**
- Header: Logo + recherche
- Filtres: Chips horizontaux
- Cards: Format standard avec badges
- Navigation: Bottom bar

3. **Detail Screen**
- Carousel d'images
- Informations clés: Grid 2x2
- Équipements: Liste avec icônes
- CTA: Fixed bottom bar

4. **Reservation Screen**
- Calendrier: Mois avec sélection de plage
- Compteurs: Adultes/Enfants avec steppers
- Résumé: Prix détaillé
- CTA: Bouton primaire

## Grille et espacements

### Système 8px

```
4px (xs) - Espacement minimal
8px (sm) - Espacement petit
16px (md) - Espacement standard
24px (lg) - Espacement large
32px (xl) - Espacement très large
48px (xxl) - Espacement section
64px (xxxl)- Espacement écran
```

### Border Radius

```
4px (xs) - Petits éléments
8px (sm) - Éléments moyens
12px (md) - Cards, inputs
16px (lg) - Cards principales
24px (xl) - Modals
9999px (full) - Boutons, chips
```

## Animations de référence

### Transitions

- **Page**: Slide depuis la droite (300ms)
- **Modal**: Fade + scale (400ms)
- **Bottom Sheet**: Slide depuis le bas (300ms)

### Micro-interactions

- **Bouton press**: Scale 0.98 (100ms)
- **Card hover**: Scale 1.02 (200ms)
- **Input focus**: Border 1px → 2px (200ms)

### Loading

- **Shimmer**: 1.5s loop
- **Skeleton**: Structure de contenu
- **Spinner**: Circular progress

## Responsive

### Breakpoints

```
Mobile: < 428px
Tablette: 428px - 768px
Desktop: > 768px
```

### Adaptations

- **Mobile**: Layout vertical, navigation bottom
- **Tablette**: Layout adaptatif, plus d'espace
- **Desktop**: Layout horizontal, navigation sidebar

## Dark Mode

### Principes

- **Cohérence**: Même identité visuelle
- **Contraste**: Respect WCAG 2.1 AA
- **Confort**: Tons plus doux
- **Identité**: Couleurs primaires reconnaissables

### Couleurs adaptées

```
Background: #152210 (Vert très foncé)
Surface: #2A3F29 (Vert moyen foncé)
Primary: #4A7A4A (Plus clair que light mode)
Text: #E0E0E0 (Presque blanc)
```

---

## Points d'attention

### À éviter

- Trop de couleurs différentes
- Animations trop rapides ou trop lentes
- Contraste insuffisant
- Éléments trop petits sur mobile

### À privilégier

- Cohérence visuelle
- Espacement généreux
- Hiérarchie claire
- Feedback utilisateur
- Accessibilité
