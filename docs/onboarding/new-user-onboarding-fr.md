# Onboarding Nouvel Utilisateur - Campbnb Québec

Bienvenue sur Campbnb Québec ! Ce guide vous accompagne lors de votre première utilisation de la plateforme.

## Objectif

À la fin de cet onboarding, vous serez prêt à:
- Naviguer dans l'application
- Rechercher des campings
- Effectuer votre première réservation
- Utiliser toutes les fonctionnalités principales

---

## Étape 1: Télécharger et Installer

### 1.1 Téléchargement

**iOS:**
1. Ouvrez l'App Store
2. Recherchez "Campbnb Québec"
3. Appuyez sur "Obtenir"

**Android:**
1. Ouvrez Google Play Store
2. Recherchez "Campbnb Québec"
3. Appuyez sur "Installer"

### 1.2 Première Ouverture

1. Appuyez sur l'icône Campbnb Québec
2. L'écran de bienvenue s'affiche

---

## Étape 2: Créer un Compte

### 2.1 Choisir la Méthode

Sur l'écran de bienvenue:
- **S'inscrire**: Créer un nouveau compte
- **Se connecter**: Si vous avez déjà un compte

### 2.2 Inscription

**Option 1: Email**
1. Appuyez sur "S'inscrire"
2. Sélectionnez "Email"
3. Entrez votre email
4. Créez un mot de passe sécurisé
5. Appuyez sur "Créer un compte"

**Option 2: Google**
1. Appuyez sur "Continuer avec Google"
2. Sélectionnez votre compte Google
3. Autorisez les permissions

**Option 3: Apple (iOS)**
1. Appuyez sur "Continuer avec Apple"
2. Utilisez Face ID ou Touch ID
3. Confirmez

### 2.3 Vérification Email

Si vous vous êtes inscrit par email:
1. Vérifiez votre boîte email
2. Ouvrez l'email de Campbnb Québec
3. Cliquez sur le lien de vérification
4. Retournez dans l'application

---

## Étape 3: Compléter votre Profil

### 3.1 Informations de Base

L'application vous guide pour compléter votre profil:

1. **Photo de profil**
- Prenez une photo ou choisissez-en une
- Une photo claire augmente la confiance

2. **Prénom et nom**
- Entrez votre nom complet
- Visible par les hôtes lors des réservations

3. **Téléphone** (optionnel mais recommandé)
- Facilite la communication avec les hôtes
- Pour les notifications importantes

4. **Bio** (optionnel)
- Présentez-vous brièvement
- Ex.: "Passionné de camping et de nature"

### 3.2 Préférences

Configurez vos préférences:
- **Langue**: Français ou English
- **Notifications**: Activez les notifications importantes
- **Thème**: Clair ou sombre

---

## Étape 4: Découvrir l'Application

### 4.1 Tour Guidé

Lors de votre première connexion, un tour guidé vous présente:

** Accueil**
- Campings recommandés pour vous
- Découvertes personnalisées
- Campings populaires

** Recherche**

- Filtres avancés

**️ Carte**

### 4.2 Navigation

**Barre de navigation inférieure:**
- 5 onglets principaux

- Indicateurs de notifications

---

## Étape 5: Première Recherche

### 5.1 Explorer l'Accueil

1. Sur l'écran d'accueil, explorez les campings recommandés
2. Appuyez sur un camping pour voir les détails
3. Familiarisez-vous avec l'interface

### 5.2 Essayer la Recherche

1. Appuyez sur l'onglet **"Recherche"**
2. Tapez une destination: Ex. "Québec"
3. Explorez les résultats
4. Utilisez les filtres pour affiner

### 5.3 Recherche Intelligente

Testez la recherche intelligente avec IA:
1. Appuyez sur "Recherche intelligente"
2. Tapez: "Camping au bord d'un lac pour la pêche"
3. Découvrez les suggestions personnalisées

---

## ️ Étape 6: Explorer la Carte

### 6.1 Accéder à la Carte

1. Appuyez sur l'onglet **"Carte"**
2. Les campings s'affichent sur la carte
3. Explorez différentes régions

### 6.2 Interagir avec la Carte

- **Zoomer**: Pincez pour zoomer
- **Déplacer**: Glissez pour explorer
- **Marqueurs**: Appuyez sur un marqueur pour voir les détails
- **Filtres**: Utilisez les filtres directement sur la carte

---

## Étape 7: Découvrir les Fonctionnalités

### 7.1 Messagerie

1. Appuyez sur l'onglet **"Messages"**
2. Découvrez l'interface de messagerie
3. Vous pourrez contacter les hôtes depuis les annonces

### 7.2 Profil

1. Appuyez sur l'onglet **"Profil"**
2. Explorez vos options:
- Mes réservations
- Favoris
- Paramètres
- Devenir hôte

### 7.3 Paramètres

1. Allez dans Profil > Paramètres
2. Explorez les options:
- Notifications
- Confidentialité
- Langue
- Sécurité

---

## Étape 8: Premiers Pas Recommandés

### 8.1 Ajouter des Favoris

1. Explorez quelques campings
2. Appuyez sur l'icône ️ pour ajouter aux favoris
3. Retrouvez-les dans Profil > Favoris

### 8.2 Contacter un Hôte

1. Ouvrez un camping qui vous intéresse
2. Appuyez sur "Contacter l'hôte"
3. Envoyez un message de test
4. Familiarisez-vous avec la messagerie

### 8.3 Configurer les Notifications

1. Allez dans Paramètres > Notifications
2. Activez les notifications importantes:
- Nouvelles demandes de réservation
- Messages
- Confirmations
- Rappels
