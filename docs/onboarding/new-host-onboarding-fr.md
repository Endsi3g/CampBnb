# ️ Onboarding Nouvel Hôte - Campbnb Québec

Bienvenue dans le programme hôte de Campbnb Québec ! Ce guide vous accompagne pour devenir hôte et créer votre première annonce.

## Objectif

À la fin de cet onboarding, vous serez prêt à:
- Activer le mode hôte
- Configurer votre profil hôte
- Créer votre première annonce
- Gérer vos réservations

## Prérequis

- Avoir un compte Campbnb Québec (invité)
- Avoir un espace de camping disponible
- Des photos de votre camping
- Environ 45 minutes

---

## Étape 1: Activer le Mode Hôte

### 1.1 Accéder aux Paramètres

1. Ouvrez l'application Campbnb Québec
2. Connectez-vous à votre compte
3. Appuyez sur l'onglet **"Profil"**
4. Appuyez sur **"Paramètres"**

### 1.2 Découvrir le Mode Hôte

1. Dans les paramètres, trouvez **"Devenir hôte"**
2. Lisez les informations sur le fait d'être hôte
3. Découvrez les avantages:
- Générer des revenus
- Rencontrer des voyageurs
- Partager votre amour du camping

### 1.3 Activer

1. Appuyez sur **"Activer le mode hôte"**
2. Confirmez votre choix
3. Le processus d'onboarding hôte commence

---

## Étape 2: Vérification du Compte

### 2.1 Vérification Email

Si votre email n'est pas encore vérifié:
1. Vérifiez votre boîte email
2. Cliquez sur le lien de vérification
3. Retournez dans l'application

### 2.2 Vérification d'Identité (Recommandé)

Pour augmenter la confiance:
1. Allez dans Paramètres > Sécurité
2. Appuyez sur "Vérifier mon identité"
3. Suivez les instructions
4. Téléchargez les documents requis

### 2.3 Numéro de Téléphone

1. Ajoutez votre numéro de téléphone
2. Vérifiez-le via SMS
3. Facilite la communication avec les invités

---

## Étape 3: Compléter le Profil Hôte

### 3.1 Informations de Base

**Nom complet**
- Vérifiez que votre nom est correct
- Visible par les invités

**Bio hôte**
Rédigez une présentation accueillante:
- "Passionné de camping depuis 20 ans"
- "J'aime partager mon amour de la nature"
- "Accueil chaleureux garanti"

**Photo de profil**
- Assurez-vous d'avoir une photo claire
- Une photo professionnelle augmente la confiance

### 3.2 Informations de Contact

**Téléphone**
- Ajoutez votre numéro
- Pour les communications urgentes

**Email**
- Vérifiez que votre email est correct
- Pour les notifications importantes

### 3.3 Informations Bancaires

Pour recevoir vos paiements:
1. Appuyez sur "Ajouter un compte bancaire"
2. Entrez vos informations:
- Numéro de compte
- Code de transit
- Nom de la banque
3. Vérifiez les informations
4. Appuyez sur "Enregistrer"

**Note**: Les paiements sont sécurisés via Stripe.

---

## Étape 4: Créer votre Première Annonce

### 4.1 Accéder à la Création

1. Depuis votre profil, appuyez sur **"Tableau de bord hôte"**
2. Appuyez sur **"Créer une annonce"**
3. Le formulaire de création s'affiche

### 4.2 Informations de Base

**Titre**
- Créez un titre accrocheur
- Maximum 60 caractères
- Ex.: "Camping au bord du lac avec vue imprenable"

**Description**
- Décrivez votre camping en détail
- Minimum 100 caractères
- Utilisez le bouton "Générer avec IA" pour une description optimisée

**Type de camping**
- Sélectionnez: Tente, VR, Prêt-à-camper, ou Sauvage

### 4.3 Localisation

1. Entrez votre adresse complète
2. Vérifiez l'emplacement sur la carte
3. Ajustez si nécessaire

### 4.4 Capacité et Équipements

**Capacité**
- Nombre maximum d'invités
- Nombre de chambres, lits, salles de bain

**Équipements**
- Cochez les équipements disponibles
- WiFi, Douche, Toilettes, Foyer, etc.

### 4.5 Règles et Politiques

**Règles de la maison**
- Rédigez les règles à respecter
- Ex.: "Pas de bruit après 22h"

**Politique d'annulation**
- Choisissez: Flexible, Modérée, ou Stricte
- **Conseil**: Commencez par "Flexible"

**Heures de check-in/check-out**
- Définissez les heures d'arrivée et de départ

### 4.6 Tarification

**Prix de base**
- Fixez votre prix par nuit
- Analysez la concurrence

**Frais supplémentaires**
- Frais de nettoyage (optionnel)
- Frais de service (calculé automatiquement)

**Réductions**
- Réduction hebdomadaire
- Réduction mensuelle

### 4.7 Photos

**Photo de couverture**
- Choisissez votre meilleure photo
- Apparaîtra dans les résultats de recherche

**Galerie**
- Ajoutez minimum 5 photos
- Photos des équipements
- Photos de l'environnement

### 4.8 Calendrier

1. Configurez votre disponibilité
2. Bloquez les dates indisponibles
3. Définissez les règles de disponibilité

---

## Étape 5: Publier votre Annonce

### 5.1 Prévisualisation

1. Appuyez sur "Prévisualiser"
2. Vérifiez toutes les informations
3. Corrigez si nécessaire

### 5.2 Publication

1. Quand tout est correct, appuyez sur **"Publier"**
2. Votre annonce est soumise pour modération
3. Vous recevrez une notification une fois approuvée

**Délai de modération**: Généralement 24-48h

---

## Étape 6: Après la Publication

### 6.1 Statut de l'Annonce

Votre annonce peut avoir différents statuts:
- **En attente**: En cours de modération
- **Active**: Visible et réservable
- **Inactive**: Temporairement désactivée

### 6.2 Première Demande

Une fois votre annonce active:
1. Vous recevrez une **notification** pour chaque demande
2. Vous avez **24h** pour répondre
3. **Acceptez ou refusez** selon votre disponibilité

### 6.3 Tableau de Bord

Explorez votre tableau de bord:
- **Vue d'ensemble**: Statistiques et revenus
- **Annonces**: Gérer vos annonces
- **Réservations**: Gérer les réservations
- **Messages**: Conversations avec les invités

---

## Conseils pour Réussir

### Pour Attirer plus d'Invités

1. **Photos exceptionnelles**: Investissez dans la photographie
2. **Description détaillée**: Soyez précis et accueillant
3. **Prix compétitifs**: Analysez le marché
4. **Réponse rapide**: Répondez dans les 2h
5. **Disponibilité**: Gardez votre calendrier à jour

### Pour Offrir une Excellente Expérience

1. **Préparez le camping** avant l'arrivée
2. **Communiquez clairement** les instructions
3. **Soyez disponible** pendant le séjour
4. **Respectez vos engagements**
