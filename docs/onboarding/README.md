## Parcours d'Onboarding

### Utilisateur Invité

1. **Créer un compte** → [Onboarding Utilisateur](new-user-onboarding-fr.md)
2. **Découvrir l'application** → Explorer les fonctionnalités
3. **Première réservation** → [Tutoriel Première Réservation](../tutorials/first-reservation-fr.md)

### Utilisateur Hôte

1. **Activer le mode hôte** → [Onboarding Hôte](new-host-onboarding-fr.md)
2. **Créer une annonce** → [Tutoriel Devenir Hôte](../tutorials/become-host-fr.md)
3. **Gérer ses annonces** → [Tutoriel Gérer ses Annonces](../tutorials/manage-listings-fr.md)
