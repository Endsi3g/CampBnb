# ️ Tutoriel: Devenir Hôte - Campbnb Québec

Ce tutoriel vous guide étape par étape pour devenir hôte et créer votre première annonce sur Campbnb Québec.

## Objectif

À la fin de ce tutoriel, vous saurez:
- Activer le mode hôte
- Créer votre première annonce
- Configurer votre profil hôte
- Publier votre annonce

## Prérequis

- Un compte Campbnb Québec (invité)
- Un espace de camping disponible
- Des photos de votre camping
- Environ 30 minutes

---

## Étape 1: Activer le Mode Hôte

### 1.1 Accéder aux Paramètres

1. Ouvrez l'application Campbnb Québec
2. Connectez-vous à votre compte
3. Appuyez sur l'onglet **"Profil"** en bas
4. Appuyez sur **"Paramètres"**

### 1.2 Activer le Mode Hôte

1. Dans les paramètres, trouvez **"Devenir hôte"**
2. Appuyez sur **"Activer le mode hôte"**
3. Lisez les informations sur le fait d'être hôte
4. Appuyez sur **"Continuer"**

### 1.3 Vérification du Compte

Pour devenir hôte, vous devez:
1. **Vérifier votre email** (si pas déjà fait)
2. **Vérifier votre identité** (optionnel mais recommandé)
3. **Ajouter un numéro de téléphone** (recommandé)

---

## Étape 2: Compléter le Profil Hôte

### 2.1 Informations de Base

1. **Nom complet**: Vérifiez que votre nom est correct
2. **Téléphone**: Ajoutez votre numéro de téléphone
3. **Bio hôte**: Rédigez une courte présentation:
- "Passionné de camping depuis 20 ans"
- "J'aime partager mon amour de la nature"
- "Accueil chaleureux garanti"

### 2.2 Informations Bancaires

Pour recevoir vos paiements:
1. Appuyez sur **"Ajouter un compte bancaire"**
2. Entrez vos informations:
- Numéro de compte
- Code de transit
- Nom de la banque
3. Vérifiez les informations
4. Appuyez sur **"Enregistrer"**

**Note**: Les paiements sont sécurisés via Stripe.

### 2.3 Photo de Profil

1. Assurez-vous d'avoir une **photo de profil claire**
2. Une photo professionnelle augmente la confiance

---

## Étape 3: Créer votre Première Annonce

### 3.1 Accéder à la Création

1. Depuis votre profil, appuyez sur **"Tableau de bord hôte"**
2. Appuyez sur **"Créer une annonce"**
3. Le formulaire de création s'affiche

### 3.2 Informations de Base

**Titre (obligatoire)**
1. Appuyez sur le champ "Titre"
2. Créez un titre accrocheur:
- "Camping au bord du lac avec vue imprenable"
- "Emplacement tente en pleine nature"
- "Camping familial avec tous les équipements"
3. Maximum 60 caractères

**Description (obligatoire)**
1. Appuyez sur le champ "Description"
2. Décrivez votre camping en détail:
- Présentation du lieu
- Points forts
- Environnement
- Expérience offerte
3. Minimum 100 caractères

**Astuce**: Utilisez le bouton "Générer avec IA" pour créer une description optimisée automatiquement.

### 3.3 Type de Camping

Sélectionnez le type:
- **Tente**: Emplacement pour tentes
- **VR**: Emplacement pour véhicules récréatifs
- **Prêt-à-camper**: Camping avec équipements inclus
- **Sauvage**: Camping en pleine nature

### 3.4 Localisation

**Adresse complète**
1. Appuyez sur "Adresse"
2. Entrez votre adresse complète:
- Numéro et rue
- Ville
- Code postal
- Province
3. L'application géolocalise automatiquement

**Vérification sur la carte**
1. Vérifiez que l'emplacement sur la carte est correct
2. Ajustez si nécessaire en déplaçant le marqueur

**Points d'intérêt**
1. Mentionnez les attractions à proximité:
- Lacs, rivières
- Sentiers de randonnée
- Villes proches
- Activités disponibles

---

## Étape 4: Capacité et Équipements

### 4.1 Capacité

1. **Nombre maximum d'invités**: Sélectionnez (1-20)
2. **Nombre de chambres**: Si applicable (0-10)
3. **Nombre de lits**: Sélectionnez (1-20)
4. **Nombre de salles de bain**: Sélectionnez (0-5)

### 4.2 Équipements

Cochez les équipements disponibles:
- **WiFi**: Connexion Internet
- **Douche**: Douche disponible
- **Toilettes**: Toilettes disponibles
- **Foyer/Fire pit**: Foyer pour feux
- **Eau potable**: Eau potable disponible
- **Électricité**: Prise électrique
- **Stationnement**: Place de stationnement
- **Animaux acceptés**: Si vous acceptez les animaux

Ajoutez d'autres équipements si nécessaire.

---

## Étape 5: Règles et Politiques

### 5.1 Règles de la Maison

Rédigez les règles à respecter:
- "Pas de bruit après 22h"
- "Respecter la nature et l'environnement"
- "Fumer uniquement dans les zones désignées"
- "Animaux en laisse obligatoire"

### 5.2 Politique d'Annulation

Choisissez votre politique:
- **Flexible**: Remboursement complet jusqu'à 24h avant
- **Modérée**: Remboursement complet jusqu'à 5 jours avant
- **Stricte**: Remboursement de 50% jusqu'à 7 jours avant

**Conseil**: Commencez par "Flexible" pour attirer plus d'invités.

### 5.3 Heures de Check-in/Check-out

1. **Heure de check-in**: Ex. 15:00
2. **Heure de check-out**: Ex. 11:00
3. Indiquez si vous êtes flexible

---

## Étape 6: Tarification

### 6.1 Prix de Base

1. Appuyez sur "Prix par nuit"
2. Entrez votre prix: Ex. 50.00 $
3. **Conseil**: Analysez les prix de la concurrence

### 6.2 Frais Supplémentaires

**Frais de nettoyage (optionnel)**
1. Activez "Frais de nettoyage"
2. Entrez le montant: Ex. 10.00 $

**Frais de service**
- Calculé automatiquement (10% du total)
- Vous n'avez rien à configurer

### 6.3 Réductions

**Réduction hebdomadaire**
1. Activez "Réduction hebdomadaire"
2. Entrez le pourcentage: Ex. 10%
3. S'applique automatiquement pour les séjours de 7+ nuits

**Réduction mensuelle**
1. Activez "Réduction mensuelle"
2. Entrez le pourcentage: Ex. 20%
3. S'applique automatiquement pour les séjours de 30+ nuits

---

## Étape 7: Photos

### 7.1 Photo de Couverture

1. Appuyez sur "Ajouter une photo de couverture"
2. Choisissez votre **meilleure photo**
3. Cette photo apparaîtra dans les résultats de recherche

### 7.2 Galerie de Photos

Ajoutez au minimum **5 photos**:
1. **Photo de couverture**: Vue générale accrocheuse
2. **Photos des équipements**: Douche, toilettes, foyer
3. **Photos de l'environnement**: Vue, nature environnante
4. **Photos de détails**: Espaces de vie, installations

**Conseils pour de belles photos:**
- Prenez des photos en **lumière naturelle**
- Montrez les **différents angles** du camping
- Incluez des **personnes** pour donner l'échelle (optionnel)
- Évitez les photos floues ou sombres

### 7.3 Ordre des Photos

1. Réorganisez les photos par glisser-déposer
2. La première photo sera la photo de couverture
3. Les meilleures photos en premier

---

## Étape 8: Calendrier et Disponibilité

### 8.1 Configurer le Calendrier

1. Appuyez sur "Calendrier"
2. Par défaut, toutes les dates sont disponibles
3. **Bloquez les dates indisponibles**:
- Dates de maintenance
- Dates personnelles
- Dates déjà réservées ailleurs

### 8.2 Disponibilité par Défaut

Configurez:
- **Jours de la semaine disponibles**: Ex. Tous les jours
- **Heures de check-in/check-out**: Déjà configurées

---

## Étape 9: Prévisualiser et Publier

### 9.1 Prévisualisation

1. Appuyez sur "Prévisualiser"
2. Vérifiez que toutes les informations sont correctes:
- Titre et description
- Photos
- Prix
- Localisation
- Équipements

### 9.2 Corrections

Si vous trouvez des erreurs:
1. Retournez à l'étape concernée
2. Modifiez les informations
3. Revenez à la prévisualisation

### 9.3 Publication

1. Quand tout est correct, appuyez sur **"Publier"**
2. Votre annonce est soumise pour modération
3. Vous recevrez une notification une fois approuvée

**Délai de modération**: Généralement 24-48h

---

## Étape 10: Après la Publication

### 10.1 Statut de l'Annonce

Votre annonce peut avoir différents statuts:
- **En attente**: En cours de modération
- **Active**: Visible et réservable
- **Inactive**: Temporairement désactivée
- **Rejetée**: Nécessite des modifications

### 10.2 Première Réservation

Une fois votre annonce active:
1. Vous recevrez des **notifications** pour chaque demande
2. Vous avez **24h** pour répondre
3. **Acceptez ou refusez** selon votre disponibilité

### 10.3 Optimisation

Pour améliorer votre visibilité:
- **Répondez rapidement** aux demandes
- **Maintenez votre calendrier à jour**
- **Collectez des avis positifs**
- **Améliorez vos photos** si nécessaire

---

## Conseils et Bonnes Pratiques

### Pour Attirer plus d'Invités

1. **Photos exceptionnelles**: Investissez dans la photographie
2. **Description détaillée**: Soyez précis et accueillant
3. **Prix compétitifs**: Analysez le marché
4. **Réponse rapide**: Répondez dans les 2h
5. **Disponibilité**: Gardez votre calendrier à jour

### Pour Offrir une Excellente Expérience

1. **Préparez le camping** avant l'arrivée
2. **Communiquez clairement** les instructions
3. **Soyez disponible** pendant le séjour
4. **Respectez vos engagements**: Dates, prix, équipements

---

## Dépannage

### Mon annonce est rejetée

1. Vérifiez les raisons du rejet dans les notifications
2. Corrigez les problèmes identifiés
3. Soumettez à nouveau

### Je ne reçois pas de demandes

- Vérifiez que votre annonce est active
- Vérifiez vos prix (peut-être trop élevés)
- Améliorez vos photos et description
- Vérifiez votre disponibilité

### Problème de paiement

1. Vérifiez vos informations bancaires
2. Contactez le support si nécessaire
